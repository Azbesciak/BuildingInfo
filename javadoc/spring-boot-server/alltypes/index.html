cs.io.buildingInfo.location.Level

Represents single level of some building structure. Contract is that it is a container for lower-spaced objects, in this case - {#link Room} However, it can be also some other location with inherits after {#link Room}.
cs.io.buildingInfo.location.Location

Basic interface to represent any locations. It assumes that every location has a cube (given in meters^3), area (meters^2), heating (KWh) and light. Also, each location has it's own id. However, its interpretation is dependent on the consumer.
cs.io.buildingInfo.location.LocationAggregator

Upper level which contains some other locations. There is no requirement of location type. All calculations are lazy - calculated only once for all aggregated location when given is called.
cs.io.buildingInfo.location.LocationCalculationsController

Main controller responsible for giving information about locations. It require JSON input format, in most cases return single value. Methods have polimorfic inputs - however, correct json structure is required, otherwise may be wrongly parsed.
cs.io.buildingInfo.location.LocationDeserializer

Json Deserializer which parses input to some location, depending on input structure. It can return {#link Room}, {#link Level} or {#link Building}. Also, when data is malformed, can throw Jackson's Parse Error/NullPointer when some fields are missing. When additional parameters are passed, they are not considered.
cs.io.buildingInfo.location.Room

Basic implementation of {#link Location}. Requires each value to be positive, otherwise throws {#link IllegalStateException}.
